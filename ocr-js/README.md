# Optical Character Recognition (OCR) System

This project is an OCR-based Invoice Processing System built using Node.js, Tesseract.js, and Sharp.

## Features
- Extracts text from invoice and bill images
- Image preprocessing using Sharp
- OCR text recognition using Tesseract.js
- Generates output in JSON and TXT formats
- Parses invoice details automatically

## Technologies Used
- Node.js
- Tesseract.js
- Sharp

## How to Run

Install dependencies:

bash  npm install 

Run OCR:

bash node ocr.js 

Run invoice parser:

bash node parse-invoice.js
